# 四柱推命パーソナルツール

単一HTMLで動く四柱推命パーソナルツールです。
公開用の本体はリポジトリ直下の `index.html` にあり、この階層の `index.html` は旧URLからの転送用です。

## 使い方

1. `index.html` をブラウザで開く
2. `占う` タブで生年月日を入力
3. 直接入力または `今の年齢 + 誕生日` の逆算入力を選ぶ
4. `命式を算出する` を押す
5. 結果は自動で履歴に保存される

## 現在入っている機能

- 年柱・月柱・日柱・時柱の算出
- 太陽黄経ベースの節入り計算
- 蔵干の `本気 / 流派式（按分）`
- 通変星・十二運・空亡・大運
- 履歴保存、検索、JSON入出力
- ライト / ダークテーマ

## メモ

- 出生時刻が不明な場合、時柱は `不明` として扱います
- 23:00以降の出生は日柱を翌日扱いで算出しています
- 節入り当日で出生時刻が不明な場合、年柱または月柱が前後する可能性があります
